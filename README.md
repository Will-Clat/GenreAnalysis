This project uses R to analyze the [Spotify Tracks Dataset](https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset) available on Kaggle.

## Getting started

1. Install/update R and RStudio to the latest versions

2. Download and run projectscript.R in RStudio

  Make sure that:
  
  - The working directory is set to the folder that contains projectscript.R
  - Within the folder above, there is a folder called "data" that contains spotify_tracks.csv
